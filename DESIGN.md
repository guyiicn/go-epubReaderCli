## 滚动机制

采用混合模式：

**单栏**: less 模式（无限滚动）
- 内容预渲染为 `[]string`（每行已按宽度换行）
- `j/k`：上下滚 1 行
- `PgDn/Space/→`：向下滚一屏（pageHeight 行）
- `PgUp/Bs/←`：向上滚一屏
- 进度 = scrollPos / totalLines

**双栏**: 分页模式（固定页大小）
- 一页 = 2 × pageHeight 行（左栏 pageHeight + 右栏 pageHeight）
- `j/k/PgDn/PgUp/Space/Bs/←/→` 全部翻整页
- scrollPos 强制对齐到页边界 (2×pageHeight 的倍数)
- 不支持按行滚动，避免左右栏内容错位
- 状态栏显示页码：`3/15页`

## 分栏

终端宽度足够时可显示两栏，用户可配置：

| 配置 columns | 行为 |
|-------------|------|
| `0`（默认） | 自动：宽度 ≥ column_threshold 时双栏，否则单栏 |
| `1` | 始终单栏 |
| `2` | 始终双栏 |

- `column_threshold` 默认 120
- `c` 键切换 1/2 栏
- 双栏时：`colWidth = (termWidth - 2) / 2`
- 翻页步长：单栏 = `pageHeight`，双栏 = `pageHeight × 2`
- 终端 resize 时重新渲染

## 核心包

### `epub/` — 解析层
- `Load(path) (*Book, error)`
- `Book.Sections []Section` — 按 spine 顺序
- `Book.TOC []TOCEntry` — 支持嵌套
- `Book.Metadata`
- **Section 标题解析**: TOC 匹配 → `<title>` → `<h1>`/`<h2>` → `Chapter N`
- **EPUB 3**: 解析 `nav.xhtml` 的 `<nav epub:type="toc">`
- `Section` = `{ID, Title, Index, HTML, CSS}`

### `render/` — 渲染层
- `Render(htmlStr, width) []string`
- 解析 HTML，提取块级元素（段落、标题、列表、引用、代码、图片占位）
- CJK 字宽换行（每个汉字可独立断行）
- 输出 `[]string`，每行已换行至指定宽度

### `store/` — 持久化层
路径：`~/.config/epub-reader/`
- `config.json` — 用户配置
- `library.json` — 书库列表
- `progress/<hash>.json` — 阅读进度
- `bookmarks/<hash>.json` — 书签
- hash = SHA256(文件路径)[:16]

## 阅读进度

- 每次翻页自动保存 `{SectionIndex, LinePos, Percent}`
- 退出时保存
- 打开书籍时自动恢复
- 全书进度 = `(已完成章节 + 当前章节百分比) / 总章节数`

## 书签

- `a` 在当前位置添加书签（可选备注）
- `B` 打开书签列表
- 选中跳转，`d` 删除
- TOC 中含书签的章节标记 `★`

## 搜索

- `/` 搜索当前章节文本
- `N` 跳到下一个匹配
- 匹配项用颜色高亮

## 配置

```json
{
  "columns": 0,
  "column_threshold": 120,
  "line_spacing": 0,
  "auto_save_interval": 0,
  "recent_books_limit": 100
}
```
- `columns`: 0=自动, 1=单栏, 2=双栏
- `line_spacing`: 行间距额外空行数
- 终端配色由用户终端控制
